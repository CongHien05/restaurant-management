Test Responsive Menu

Resize trình duyệt hoặc mở Developer Tools (F12) để test responsive:

  • Desktop (≥992px): Menu hiển thị bên trái
  • Mobile/Tablet (<992px): Bấm nút ☰ góc trên trái để mở menu
Kiểm tra:
  1. Resize browser về mobile size (<768px)
  2. Bấm nút hamburger ☰
  3. Menu slide vào từ trái
  4. Click menu item → tự động đóng
  5. Click backdrop (màn tối) → đóng menu
